# /dawa PWA icons

Placeholders only. These files were generated by
`scripts/gen-dawa-icons.mjs` — a pure-Node (zero-dependency) script that
emits solid HealiX-green squares with a blocky white "N" glyph.

Files:

- `dawa-192.png` — any-purpose, 192×192 (Android launcher, iOS home screen).
- `dawa-512.png` — any-purpose, 512×512 (splash screens, Play Store).
- `dawa-maskable-512.png` — maskable, 512×512 with ~10% safe-zone padding
  (used by Android adaptive icons).

**Before launch**, the designer should replace these with real artwork.
Respect:

- PWA maskable safe zone: keep the meaningful glyph inside the central
  80% (406×406 box inside a 512 canvas).
- Background colour `#1D9E75` (HealiX accent) by default, or `#0A0F14`
  (HealiX bg) if the designer goes for dark maskable.

Regenerate:

```bash
node scripts/gen-dawa-icons.mjs
```
